# Размещение сайта

## Вариант размещения

Для публикации можно использовать бесплатный тариф сервиса Render или PythonAnywhere. Проект является Flask-приложением и запускается командой:

```bash
python run.py
```

## Подготовка

1. Загрузить проект в публичный репозиторий GitHub.
2. Создать приложение на выбранном хостинге.
3. Указать команду установки зависимостей:

```bash
pip install -r requirements.txt
```

4. Указать команду запуска:

```bash
python run.py
```

5. После публикации проверить главную страницу, поиск, контакты и личный кабинет.

## Данные для отчета

В исходные данные отчета нужно добавить:

- ссылку на публичный репозиторий;
- ссылку на опубликованный сайт;
- логины и пароли тестовых учетных записей.
