# Размещение сайта на SpaceWeb

Для публикации используется бесплатный виртуальный хостинг SpaceWeb:
https://sweb.ru/hosting/free/

## Публикация

1. Зарегистрировать аккаунт SpaceWeb и активировать бесплатный хостинг.
2. В разделе `Хостинг` -> `Сайты` выбрать конфигурацию
   `Apache 2.4 + PHP 8.3 + mod_wsgi python3.11`.
3. Загрузить содержимое репозитория в каталог `public_html` через
   файловый менеджер. Удобнее загрузить ZIP-архив и распаковать его в текущий
   каталог.
4. Убедиться, что в `public_html` находятся `.htaccess`, `wsgi.py`, `run.py`,
   `requirements.txt`, а также каталоги `app`, `instance` и `vendor`.
5. Сохранить настройки сайта и открыть выданный адрес вида
   `https://<имя>.swtest.ru`.

SSH и установка пакетов на сервере не требуются: Flask и его зависимости уже
находятся в каталоге `vendor` и подготовлены для Linux с Python 3.11.

## Проверка

Проверить главную страницу, поиск, контакты, личный кабинет и страницу 404.
