# Отчетные материалы по практике

## Тема работы

Разработка информационного сайта на тему строительства частных домов.

## Цель работы

Создать сайт, который представляет услуги строительной компании, содержит проекты домов, статьи, новости, контакты, поиск по материалам и простую систему учетных записей пользователей.

## Используемые технологии

- Python 3.14.4;
- Flask;
- HTML;
- CSS;
- JavaScript;
- JSON-файлы для хранения учебных данных.

## Реализованные разделы

Основные страницы сайта:

1. Главная.
2. Проекты домов.
3. Услуги.
4. Статьи.
5. Новости.
6. Контакты.
7. Личный кабинет.

Дополнительные страницы:

- карта сайта;
- поиск;
- регистрация;
- страница 404.

## Реализованные функции

- вывод проектов домов;
- публикация услуг, статей и новостей;
- отдельные страницы статей;
- форма обратной связи;
- сохранение обращений пользователей;
- поиск по материалам сайта;
- стандартный режим отображения;
- режим для слабовидящих;
- вход в личный кабинет;
- роли пользователей;
- создание пользователя сотрудником;
- самостоятельная регистрация клиента.

## Учетные записи для проверки

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | admin | admin123 |
| Сотрудник | manager | manager123 |
| Клиент | client | client123 |

## Сценарий создания пользователя сотрудником

1. Открыть страницу личного кабинета.
2. Войти под учетной записью `manager` или `admin`.
3. Заполнить форму создания пользователя.
4. Выбрать роль пользователя.
5. Нажать кнопку создания.
6. Новый пользователь сможет войти через страницу личного кабинета.

## Проверка работы

Для проверки нужно запустить приложение локально и открыть сайт в браузере. Проверяются основные страницы, форма контактов, поиск, режим для слабовидящих, вход в кабинет и переход на несуществующий адрес.

## Размещение

Проект подготовлен для публикации в публичном репозитории GitHub. Для размещения сайта можно использовать бесплатный хостинг с поддержкой Python-приложений, например Render или PythonAnywhere. После публикации ссылка на сайт добавляется в исходные данные отчета.
