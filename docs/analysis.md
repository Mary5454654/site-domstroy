# Анализ предметной области

## Тема проекта

Разрабатывается информационный сайт компании, которая занимается строительством частных домов. Сайт должен показывать услуги, готовые проекты, полезные материалы и контакты для связи с клиентами.

## Представляемый продукт

На сайте представлен комплекс услуг по строительству частных домов:

- проектирование дома;
- строительство домов из газобетона, кирпича и дерева;
- подбор типового проекта;
- расчет ориентировочной стоимости;
- консультации по порядку строительства;
- сопровождение клиента от заявки до сдачи объекта.

## Границы продукта

Сайт не является полноценной CRM-системой или интернет-магазином. Он решает информационные задачи: знакомит пользователя с компанией, показывает примеры проектов, публикует статьи и новости, принимает обращения через форму связи.

## Цели сайта

- дать посетителю понятное представление об услугах компании;
- показать примеры домов и порядок работы;
- собрать первичные заявки от потенциальных клиентов;
- повысить доверие за счет статей, новостей и контактной информации;
- обеспечить удобную навигацию и поиск по материалам сайта.

## Целевая аудитория

Основная аудитория сайта:

- частные лица, планирующие строительство дома;
- семьи, выбирающие проект для постоянного проживания;
- владельцы земельных участков;
- пользователи, сравнивающие строительные компании;
- сотрудники компании, которые могут обрабатывать обращения и создавать пользователей.

## Ожидаемые помехи

- пользователь не знает, какой тип дома выбрать;
- сложно быстро оценить примерную стоимость;
- посетители не доверяют компаниям без портфолио и понятных контактов;
- большой объем информации может мешать поиску нужного материала.

## Что дает сайт

Сайт помогает структурировать информацию о компании и услугах. Посетитель может посмотреть проекты, прочитать статьи, найти новости, отправить сообщение и понять, как начать работу с компанией.

## Анализ похожих сайтов

При анализе сайтов строительных компаний можно выделить удачные решения:

- крупный первый экран с понятным предложением;
- отдельные страницы проектов и услуг;
- фотографии домов и краткие характеристики;
- форма обратной связи на странице контактов;
- статьи с советами по выбору материалов и планировок;
- версия для слабовидящих или элементы доступности.

В разрабатываемом сайте используются эти решения, но без сложной архитектуры и без CMS.

## Структура сайта

Планируется 7 основных страниц:

1. Главная.
2. Проекты домов.
3. Услуги.
4. Статьи.
5. Новости.
6. Контакты.
7. Личный кабинет.

Дополнительно предусмотрены карта сайта, страница поиска и страница ошибки 404.
