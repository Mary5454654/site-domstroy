# План продвижения сайта

## Цели продвижения

- привлечь пользователей, которые планируют строительство частного дома;
- повысить доверие к компании;
- увеличить количество заявок через форму обратной связи;
- улучшить видимость сайта в поисковых системах.

## Основные действия

1. Заполнить страницы понятными текстами по темам строительства, материалов, сметы и выбора проекта.
2. Подготовить заголовки страниц с ключевыми словами: строительство частных домов, проекты домов, строительство под ключ.
3. Регулярно публиковать новости и статьи.
4. Добавить сайт в карты и справочники организаций.
5. Разместить ссылку на сайт в социальных сетях компании.
6. Использовать фотографии готовых объектов и отзывы клиентов.
7. Проверять обращения через форму связи и быстро отвечать пользователям.

## Контент для продвижения

Для продвижения лучше всего подходят:

- статьи с практическими советами;
- сравнение материалов;
- обзоры проектов домов;
- новости о выполненных объектах;
- ответы на частые вопросы клиентов.

## Оценка результата

Эффективность продвижения можно оценивать по количеству посещений сайта, числу заявок, переходам из поиска и активности пользователей на страницах статей и проектов.
